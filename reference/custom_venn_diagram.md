# Comparing Overlap of two GRange Objects

Comparing Overlap of two GRange Objects

## Usage

``` r
custom_venn_diagram(conditionA, conditionB, DataA, DataB)
```

## Arguments

- conditionA:

  String with the name of the DataA

- conditionB:

  String with the name of DataB

- DataA:

  GRanges Object

- DataB:

  GRanges Object

## Value

GGplot \| Venn Diagram showing overlaps between two GRanges Objects.
