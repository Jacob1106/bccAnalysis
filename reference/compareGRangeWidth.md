# Width Comparison of 2 GRanges Objects

Width Comparison of 2 GRanges Objects

## Usage

``` r
compareGRangeWidth(Obj1, Obj2)
```

## Arguments

- Obj1:

  GRanges Object

- Obj2:

  GRanges Object

## Value

GGplot \| Returns a plot that shows the width of Object 1 and Object 2.
Plot = (x = Width of Peak) (y = Density of a particular width (How many
peaks have that width))
